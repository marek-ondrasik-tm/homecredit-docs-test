Když chcete poskytnout zákazníkovi (který chce zaplatit za nákup úvěrem)
možnost vybrat si z dostupných splátkových programů, když je ještě na vašem
webových stránkách, je třeba zavést "splátkovou kalkulačku".

Pro integraci splátkové kalkulačky do svých webových stránek zvolte jeden z následujících přístupů:

- A) Implementace vlastní splátkové kalkulačky s vaší FE ve vašem designu, která využívá určené zdroje API ([popsáno níže](#implementing-own-installments-calculator)).


- B) Integrace javascriptového widgetu společnosti Home Credit (popsaného v [zvláštní kapitole](../Reference/java-script-widget-integration)), který:
  - poskytuje grafické uživatelské rozhraní a funkce front-endu
  - zpracovává všechny potřebné výpočty
  - poskytuje potřebné údaje back-endu e-shopu, aby je bylo možné snadno použít v rámci [požadavku na vytvoření *aplikace*](./create-using-post-3)

> POZNÁMKA: Implementace splátkové kalkulačky je volitelná. Zákazníci mohou vždy používat splátkovou kalkulačku zabudovanou ve front-endu Home Creditu.


## Implementace vlastní splátkové kalkulačky

K implementaci vlastní splátkové kalkulačky (s vlastním návrhem grafického uživatelského rozhraní) budete potřebovat následující zdroje:

- `/financing/v1/installmentProgramsOffers` - pro zjištění dostupných nabídek splátkových programů a jejich podrobností

- `/financing/v1/installmentPrograms` - pro zjištění podrobností vybraného splátkového programu

- `/financing/v1/installmentProgramsDownPayments` - zjištění možností akontace v rámci dostupných nabídek splátkových programů

### Nabídky splátkových programů

Když chce zákazník zaplatit nákup prostřednictvím úvěru a vy chcete
mu zobrazit nabídku dostupných splátkových programů,
použijte zdroj `/financing/v1/installmentProgramsOffers` se správně nastavenými nabídkami
vyplněným požadavkem, který obsahuje alespoň následující údaje:

- Kód produktové sady (správný kód produktové sady byste měli znát ze seznamu produktů)
Obchodního zástupce Home Creditu nebo jiného podobného zdroje;
pokud kód produktové sady neuvedete, bude použita vaše výchozí produktová sada)
- Cena nákupu zákazníka

Pokud byl požadavek úspěšně zpracován, obdržíte stavový kód HTTP 200
a pole objektů dostupných splátkových programů s minimálně
následujícími informacemi v odpovědi:

- Splátkový program (pole)
  - Kód produktu splátkového programu
  - Typ splátkového programu (v případě nabídky pouze jednoho splátkového programu)
  v odpovědi je to typ `FAVOURITE`)
  - Výše splátky
  - Počet splátek

V ostatních případech se zobrazí jedna z následujících chyb:

| Stavový kód | Kód | Vysvětlení |
| ----------- | --------------- | -------------------- |
| 400 | INVALID_REQUEST | Požadavek nebyl správně naformátován (chybná syntaxe požadavku, příliš velká velikost atd.) |
| 422 | UNPROCESSABLE | Požadavek byl dobře zformulován, ale nebylo možné jej následovat kvůli sémantickým chybám |
| 500 | INTERNAL_SERVER_ERROR | Nastal neočekávaný stav a není vhodná žádná konkrétnější zpráva |

Výpočet nabídky splátkových programů podle zadaného vstupu

| funkce | podporováno zdrojem |
| ------- | --------------------- |
| stránkování | ne |
| třídění | ne |
| filtrování | ne |

### Splátkové programy

Když chce zákazník zjistit podrobnosti o jednom z dříve nabízených
splátkových programů, použijte zdroj `/financing/v1/installmentPrograms`
s řádně vyplněným požadavkem, který obsahuje alespoň následující údaje:

- Kód produktu daného splátkového programu
- Cena nákupu zákazníka

Pokud byl požadavek úspěšně zpracován, zobrazí se stav HTTP
kód 200 a objekt s informacemi o zadané splátce
programu s minimálně následujícími informacemi v odpovědi:

- Typ splátkového programu
- Výše splátky
- Počet splátek
- Podrobnosti o splátkovém programu

V ostatních případech se zobrazí jedna z následujících chyb:

| Stavový kód | Kód | Vysvětlení |
| ----------- | --------------- | -------------------- |
| 400 | INVALID_REQUEST | Požadavek nebyl správně naformátován (chybná syntaxe požadavku, příliš velká velikost atd.) |
| 422 | UNPROCESSABLE | Požadavek byl dobře zformulován, ale nebylo možné jej následovat kvůli sémantickým chybám |
| 500 | INTERNAL_SERVER_ERROR | Nastal neočekávaný stav a není vhodná žádná konkrétnější zpráva |

Podrobné informace o programu pro výpočet splátek. Vrátí splátkový program, který nejvíce odpovídá zadanému vstupu.

### Splátkové programy akontace

Když chce zákazník zaplatit nákup prostřednictvím úvěru a také
chce zvolit vlastní nastavení akontace v rámci nabízených splátek
programů, použijte zdroj `/financing/v1/installmentProgramsDownPayments`
s řádně vyplněným požadavkem, který obsahuje alespoň následující údaje:

- Kód produktové sady (správný kód produktové sady byste měli znát z dokumentu
Obchodního zástupce Home Creditu nebo z jiného podobného zdroje)
- Cena nákupu zákazníka

Pokud byl požadavek úspěšně zpracován, obdržíte stavový kód HTTP
200 a objekt informací o nastavení dostupných zálohových plateb
s minimálně následujícími informacemi v odpovědi:

- Minimální výše zálohy
- Maximální výše zálohové platby
- Krok mezi danou minimální a maximální výší zálohy

V ostatních případech se zobrazí jedna z následujících chyb:

| Stavový kód | Kód | Vysvětlení |
| ----------- | --------------- | -------------------- |
| 400 | INVALID_REQUEST | Požadavek nebyl správně naformátován (chybná syntaxe požadavku, příliš velká velikost atd.) |
| 422 | UNPROCESSABLE | Požadavek byl dobře zformulován, ale nebylo možné jej následovat kvůli sémantickým chybám |
| 500 | INTERNAL_SERVER_ERROR | Nastal neočekávaný stav a není vhodná žádná konkrétnější zpráva |
