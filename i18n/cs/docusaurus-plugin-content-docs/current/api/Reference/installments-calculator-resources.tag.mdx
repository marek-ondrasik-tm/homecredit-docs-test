Pokud chcete svému zákazníkovi (který chce zaplatit za nákup pomocí úvěru) nabídnout možnost vybrat si z dostupných splátkových programů, zatímco je ještě na vašem webu, je nutné implementovat "kalkulačku splátek“.

Vyberte si jeden z následujících přístupů k integraci kalkulačky splátek do vaší webové stránky:

- A) API - Implementace vlastní kalkulačky splátek s vaším FE ve vašem designu, který využívá určené API zdroje ([popis níže](#implementing-own-installments-calculator)).

- B) Widget - Integrace javascriptového widgetu Home Credit (popsaného v [speciální kapitole](./java-script-widget-integration)), který:
  - poskytuje GUI a front-endové funkce
  - zpracovává všechny potřebné výpočty
  - poskytuje potřebná data back-endu e-shopu, aby je bylo možné snadno použít v rámci [*žádosti o vytvoření aplikace](./create-application)

- C) Standalone - Home Creditem vystavená externí kalkulačka splátek, kde implementace kalkulačky splátek není povinností partnera viz [tady](/docs/documentation/dev-env#3-home-creditem-vystavená-externí-kalkulačka-splátek---standalone) 
## Implementace vlastní kalkulačky splátek

K implementaci vlastní kalkulačky splátek (s vaším designem GUI) budete potřebovat následující zdroje:

- `/financing/v3/installmentProgramsOffers` - k vyhledání dostupných nabídek splátkových programů a jejich podrobností

- `/financing/v3/installmentPrograms` - pro zjištění podrobností o vybraném splátkovém programu

- `/financing/v3/installmentProgramsDownPayments` - pro zjištění možností zálohy v rámci dostupných nabídek splátkových programů
