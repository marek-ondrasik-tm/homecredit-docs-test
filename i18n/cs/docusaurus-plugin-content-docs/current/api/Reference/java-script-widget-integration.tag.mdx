Aby bylo možné používat javascriptový widget Home Credit jako kalkulačku splátek, musí být správně nakonfigurován back-end vašeho e-shopu. Jak na to, je v současné době popsáno v [našem online repozitáři](https://github.com/homecreditcz/widget-calculator) (v současné době napsáno v češtině).



```mdx-code-block
import DocCardList from '@theme/DocCardList';
import {useCurrentSidebarCategory} from '@docusaurus/theme-common';

<DocCardList items={useCurrentSidebarCategory().items}/>
```
